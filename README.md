# 消消乐游戏 (Xiaoxiaole Game)

一个功能丰富的消消乐游戏，支持多种游戏模式和精美的视觉效果。

## 🎮 主要功能

### 游戏模式
- **经典模式**：在有限步数内达到目标分数
- **限时模式**：在限定时间内获得尽可能高的分数
- **无尽模式**：没有时间和步数限制，挑战你的极限
- **解谜模式**：完成特定的关卡任务
- **连锁风暴**：通过连续消除获得额外奖励
- **特殊方块挑战**：应对各种特殊方块的挑战
- **重力反转**：体验重力反转的特殊玩法

### 游戏特性
- 精美的游戏界面和流畅的动画效果
- 丰富的音效和背景音乐
- 响应式设计，适配不同设备
- 清晰的游戏统计和进度显示
- 游戏帮助和设置功能

## 🛠️ 技术栈

- **前端技术**：HTML5 + CSS3 + JavaScript (ES6+)
- **音频处理**：Web Audio API
- **模块化设计**：使用ES6模块化语法
- **无需构建工具**：直接运行，无需额外配置

## 🎯 游戏玩法

### 基础玩法
1. 点击相同颜色的相邻方块（横向或纵向）
2. 至少需要3个相同颜色的方块才能消除
3. 消除方块后，上方的方块会下落，新方块会从顶部补充
4. 连续消除可以获得更高的分数

### 特殊方块
- **炸弹方块**：消除周围的方块
- **火箭方块**：消除一行或一列的方块
- **彩虹方块**：可以与任何颜色的方块匹配

## 📖 游戏模式详解

### 经典模式
- 有限的步数
- 需要达到目标分数才能通关
- 每关难度递增

### 限时模式
- 限定时间（通常为60秒）
- 在时间结束前获得尽可能高的分数
- 连续消除可以获得额外时间奖励

### 无尽模式
- 没有时间和步数限制
- 随着分数提高，难度逐渐增加
- 挑战你的极限分数

### 解谜模式
- 每关有特定的任务目标
- 需要使用有限的步数完成任务
- 考验策略和思考能力

### 连锁风暴
- 目标是创建尽可能长的消除连锁
- 连锁次数越多，奖励分数越高

### 特殊方块挑战
- 游戏中会出现各种特殊方块
- 需要掌握不同特殊方块的使用技巧

### 重力反转
- 重力方向会定期反转
- 需要适应不同的重力方向来进行游戏
